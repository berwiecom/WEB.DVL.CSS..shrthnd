shrthnd

What is Shrthnd?

Shrthnd is a handy tool that converts CSS properties into shorthand, making shorter and more readable stylesheets.

How do I use it?

Simply input your CSS and shorthand syntax will be used wherever possible.

Can I contribute?

Sure, Shrthnd is open-source and can be found on GitHub.
